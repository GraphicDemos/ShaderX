Sample Overview
THis sample shows how to render complex Mathematical formulas using MS HLSL Shader language and Pixel Shader2.0.

How does the sample work?
First of all a set of complex functions is defined within MS HLSL to work with. Then 3 Textures are defined as RENDERTARGET Textures, two of them are used as in-between memory,due to the fact that most pixelShader2.0 hardware only allows to use 128 commands per pass, the result of each iteration is stored in one of the two textures.

Finally the result is read and transformed into a color value.

The program itself uses an instance of the D3DXInclude interface, to load different function at designated position in the fx file
